Text Reveal

Each line reveals with a mask,
sliding up one by one.

Just add .fx-text-reveal-st to any text element. The SDK splits it into lines, wraps each in an overflow-hidden mask, and animates them in sequence.

Bare Classes

No -pl or -st suffix needed.

Elements with bare .fx-text-reveal inside a <section> are auto scroll-triggered using the section as the trigger. Zero config needed.

Tag Map

This heading animates with zero classes.

When FX.config.tagMap is set, elements inside sections are auto-animated by their tag name. This entire section has no .fx-* classes at all.
